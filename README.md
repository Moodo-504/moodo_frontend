# 🐝 Moodo - Frontend

Moodo(무도)는 '할 일 기록(To-Do)'과 '감정 일기(Mood Diary)'를 결합한 감성 지능형 생산성 서비스입니다. 매일 할 일을 기록하고, 각 항목에 대해 자신의 기분을 선택하여 기록할 수 있습니다.

### 🛠 Tech Stack

* **Language:** JavaScript (ES6+)
* **Library:** React 18
* **Build Tool:** Vite
* **Routing:** React Router DOM v6/v7
* **CSS Framework:** Tailwind CSS
* **Auth:** React Context API (AuthContext)

---

### 💡 Moodo 주요 기능 및 개발 포인트

#### 1. To-Do & Mood Integration
* 단순한 할 일 완료 작성을 넘어, 작성 시점의 기분을 즉시 선택하여 기록하는 결합형 UX 제공.

#### 2. Emotional UI/UX
* 사용자가 선택한 오늘의 전반적인 기분에 따라 메인 테마 색상이 부드럽게 변화하는 감성 디자인 적용.

#### 3. Simple Logging & Interaction
* 복잡한 일기 작성 과정 없이, 직관적인 아이콘 클릭만으로 할 일과 감정을 동시에 기록하는 루틴 형성.

#### 4. Personal Calendar
* 기록된 감정 데이터와 할 일 달성률을 시각화하여, 캘린더로 기분과 할 일 목록을 확인.

### 🚀 시작하기

#### 1. 필수 요구사항
* **Node.js:** v18.0.0 이상
* **npm:** v9.0.0 이상

#### 2. 설치 및 실행
```bash
# 레포지토리 클론
git clone [https://github.com/Moodo-Project/Moodo-Frontend.git](https://github.com/Moodo-Project/Moodo-Frontend.git)

# 의존성 패키지 설치
npm install

# 로컬 개발 서버 실행
npm run dev
